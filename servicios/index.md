# Servicios de openXpert

Ofrecemos soluciones tecnológicas integrales adaptadas a las necesidades de tu empresa.

## Desarrollo Backend

Desarrollamos APIs REST robustas y escalables utilizando las tecnologías más modernas:

- **Node.js** - Servidores de alto rendimiento
- **NestJS** - Framework escalable para enterprise
- **Python** - Automatización y data science
- **PostgreSQL/MySQL** - Bases de datos relacionales
- **Supabase** - Backend as a Service

## Desarrollo Frontend

Creamos interfaces de usuario modernas y responsivas:

- **React** - BibliotecauiFacebook
- **React Native** - Apps móviles nativas
- **Expo/EAS** - Desarrollo rápido de apps
- **Ionic** - Apps híbridas multiplataforma

## Desarrollo Desktop

Migración y mantenimiento de sistemas legados:

- **OpenEdge / Progress RDBMS 4GL**
- **COBOL** - Sistemas empresariales
- **Visual FoxPro** - Aplicaciones legacy
- **VBA** - Automatización Office

## Infraestructura y DevOps

Administración de servidores y automatización:

- **Linux** - Debian, Ubuntu Server
- **Docker** - Contenedores
- **Bash/AWK** - Scripts de automatización
- **SSH** - Administración remota

## Consultoría ERP

Implementación de sistemas ERP para optimizar procesos empresariales:

- Integración de módulos financieros
- Gestión de inventario y logística
- Control de producción
- Reporting y analytics
